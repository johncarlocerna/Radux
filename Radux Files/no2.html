Radux is a website that integrates real-time news with social media. It does this through a frequently refreshing RSS program.

  Radux also features a self reported news section where a user can post photos and a short description of current events in their area.

  Future advancements of this service will include a mobile version that will allow users to export posts to social media outlets.










  This program was developed by Chris Hoyek, Johncarlo Cerna, Andre Simoes, Sebastian Conde, and Nishant Thomas: October 16-17, 2015.

  This project utilized QCAD, WebGLEarth, Balsamiq, Microsoft Office Powerpoint, and RSSinclude.
